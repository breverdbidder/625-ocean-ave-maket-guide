# 625 Ocean Avenue — Maket.ai Prompt Engineering Guide

## Overview
Interactive single-page HTML reference guide for generating accurate AI renders of **625 Ocean Street, Satellite Beach, FL** using [Maket.ai](https://maket.ai). Built around the official EDC Engineering & Design Concepts floor plans (Sheets A1–A5).

## Features
- **Anti-Hallucination Protocol** — 6 rules to prevent AI fabrication in renders
- **Project Data Sheet** — Ground truth tables for building envelope, floor areas, room schedule, and window schedule
- **Maket.ai Workflow** — Step-by-step feature map (Plan Recognizer → Editor → 3D → Render → Visualizer)
- **Copy-to-Clipboard Prompts** — Pre-written prompts for floor plans, interiors, exteriors, and room-by-room renders
- **Material & Furniture Specs** — Marble assignments, furniture style guide, and exclusion lists
- **Interactive Quality Checklist** — Click-to-check validation for structural accuracy, interior accuracy, and style consistency
- **Master Prompt Formula** — 8-part template for writing zero-hallucination prompts
- **Common Mistakes & Fixes** — Table of 10 common errors with corrected prompt examples

## Usage
1. Open `index.html` in any modern browser
2. Use the Table of Contents to navigate sections
3. Click **Copy** on any prompt box to copy it to clipboard
4. Paste into Maket.ai's Advanced/Prompt mode
5. Upload the corresponding PDF floor plan page as reference image
6. Run the Quality Checklist after each render
