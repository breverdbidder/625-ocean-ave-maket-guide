0. Anti-Hallucination Protocol

CRITICAL: These Rules Prevent AI Fabrication

AI rendering tools (Nano Banana Pro, Midjourney, DALL-E) will invent rooms, add windows, change proportions, and fabricate measurements unless you constrain them with EXACT data. Follow these rules:

RULE 1: Never Use Vague Language

BAD: "large living room" → GOOD: "30 ft x 22 ft rec room, 1,966 SF total first floor"

BAD: "beach house style" → GOOD: "3-story coastal modern, CMU block, stucco exterior, flat roof with parapet"

RULE 2: Always Anchor to Engineering Drawings

Every prompt must reference the actual sheet number (A1-A5) and room from the EDC plans. Upload the PDF page as reference image BEFORE writing the text prompt.

RULE 3: Specify What Does NOT Exist

Tell the AI what to exclude: "NO pool on ground level, NO pitched roof, NO bay windows, NO second-floor balcony on east elevation." AI adds features by default — you must block them.

RULE 4: Lock Room Count & Dimensions

State exact count: "EXACTLY 3 bedrooms, EXACTLY 4 bathrooms, EXACTLY 2 wet bars." AI will add or remove rooms if you don't lock them.

RULE 5: One Room Per Prompt for Interiors

Never ask for "render the whole floor." Always: one room, one angle, one prompt. This prevents the AI from merging or inventing spaces.

RULE 6: Validate Every Output Against Plans

After EVERY generation, cross-check: window count, door placement, room proportions, ceiling height. If anything deviates from the EDC plans, regenerate with corrections — NEVER accept hallucinated details.

9. Master Prompt Structure Formula

Every prompt you write for Nano Banana Pro should follow this exact 8-part structure. Items listed first carry the most weight.

1. SCENE TYPE

"Interior rendering of..." / "Exterior elevation of..." / "Aerial view of..."

2. ROOM + DIMENSIONS

Room name, floor level, square footage, key measurements from plans

3. ARCHITECTURAL FEATURES

Windows (count + size), doors, fireplace, stairs, soffits — from EDC sheets

4. MATERIALS + FINISHES

Specific marble name + color description, wall finish, ceiling treatment

5. FURNITURE + FIXTURES

Specific pieces, colors, materials, placement relative to room features

6. LIGHTING

Natural light source + direction, artificial fixtures, time of day, mood

7. CAMERA + QUALITY

Angle, lens, perspective, "photorealistic, 8K, Architectural Digest quality"

8. EXCLUSIONS (CRITICAL)

"EXCLUDE: No [list everything you DON'T want]" — this is your hallucination firewall

Quick Template (Copy & Modify)

The #1 Anti-Hallucination Rule

If it's not in the EDC engineering drawings (Sheets A1-A5), it does NOT exist in the house. Period. Every number, every window, every door, every room must trace back to a specific sheet and callout. If the AI generates something you can't find in the plans, it hallucinated it — regenerate with a correction.
